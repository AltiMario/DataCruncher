SeerDataCruncher

is a Data Quality Firewall, Data Quality Monitor and ETL middleware to manage data streams on the fly.

View project on GitHub

The problem

Millions of data streams are coming in your system and you realized there is too much garbage inside.

The solution

Instead of cleaning your db when it's too late, why don't you block the bad data before they occur? SeerDataCruncher is built to do that. Protect the quality of your data in real-time.

How to use

After Installation, you have to configure and certify the source data, the data type and the values expected using a simple web interface (no technical skill required). Like a network firewall, when it's up and running you don't know what is going on in background but with SeerDataCruncher you have an amazing dashboard to manage and view everything.

Authors

In 2015, Alfredo Gemma and Mario Altimari founded see-r.com, where SeerDataCruncher is used and evolved. We decided to offer the community this "piece" of technology, hoping it will be useful for everyone.
